To declare X as a float, use the syntax

float X;

This will give you about 6 place accuracy for your numbers. For more accuracy use double. Float is read or written with the %f data specification.

%f12.1234566 digit default
%9.6f12.123456just fits
%5.2f12.12just fits
%8.2f___12.12right justified
%3.2f12.12left justified
