Editor Commands
Turbo C version 3.0

Block Commands:

There are at least three different ways to select a block of text.

1.  Drag your mouse over the text you want to select.
	
2.	a)  Place the cursor at the beginning of the text you wish to select.
	b)  Hold down the shift key.
	c)  Move the cursor to the end of the text you wish to select while 
	holding down the shift key.

3.	a)  Place the cursor at the beginning of the text you wish to select.
	b)  Press < CTRL+K > < B > .
	c)  Move the cursor to the end of the text you wish to select.
	d)  Press < CTRL+K > < K >.

You may use the following commands after a block has been selected:

	Copy block			< CTRL+K > < C >
	Move block			< CTRL+K > < V >
	Delete block			< CTRL+K > < Y >
	Print block			< CTRL+K > < P >
	Write block to disk		< CTRL+K > < W >
	Read block from disk		< CTRL+K > < R >
	Move to beginning of block	< CTRL+Q > < B >
	Move to end of block		< CTRL+Q > < K >
	Hide/Show block (toggle)	< CTRL+K > < H >
	Indent block			< CTRL+K > < I >
	Unindent block			< CTRL+K > < U >
	Copy to clipboard		< CTRL+INS > 
	Cut to clipboard		< SHIFT+DEL > 
	Paste from clipboard		< SHIFT+INS > 

	Cursor Commands

	Page up				< PgUp >
	Page down			< PgDn >
	Beginning of line		< HOME >
	End of line			< END >
	Top of window			< CTRL+HOME > 
	Bottom of window		< CTRL+END >
	Top of File			< CTRL+PgUp >
	Bottom of file			< CTRL+PgDn >
	Move to previous position	< CTRL+Q > < P > 

	Delete line			< CTRL+Y >
	Insert new line			< CTRL+N > 

	Set Place Marker   < CTRL+K > < n >  - where n  is from 0-9
	Find place marker  < CTRL+K > < n >  - where n  is from 0-9
