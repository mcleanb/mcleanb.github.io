char is a data type for a variable that holds exactly one character. A charcacter constant is 'one' symbol between single quotes. The syntax for declaring a char follows.
char first, middle, last, newline;
first='T' ; middle='B' ; last = 'M' ; newline = '\n' ;
Use the %c conversion specification to read or write a char with scanf or printf. You can also use the macros, getchar() or putchar() without using the %c.
scanf("%c", &last); is the same as last = getchar();
printf("%c", first); is the same as putchar(first);
