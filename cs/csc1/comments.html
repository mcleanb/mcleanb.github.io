Detailed comments must be written throughout a program to remind oneself or one's successor of every detail about the program.
In C++ or JAVA, you can start a comment with /* ... and end it with */ or one line in C++, or Java can be commented by starting it with // .
In BASIC, you can start a comment with

REM

or you can finish a line of code by using a single quote.

REM
REM This program was entered on 4/2/97
REM
LET F = 212 ' This is input data.
LET C = 5/9*(F-32) ' This is the formula for conversion to Celsius.

PRINT F; "degrees F =";C;"degrees C." ' this shows a PRINT.
END
